An interactive Microsoft Excel dashboard designed to analyze Emergency Room (ER) operations and provide actionable insights into patient flow, waiting times, admissions, referrals, and overall hospital performance.

---

## 📌 Project Overview

Hospitals generate large amounts of patient data every day. This dashboard transforms raw emergency room data into meaningful visual insights, enabling healthcare administrators to monitor key performance indicators (KPIs), identify operational bottlenecks, and support data-driven decision-making.

---

## 🎯 Objectives

- Analyze emergency room patient trends
- Monitor patient waiting times
- Track admission and referral rates
- Evaluate patient demographics
- Identify peak visiting periods
- Improve operational efficiency through data visualization

## 📂 Dataset

The dashboard is built using hospital emergency room data containing patient demographics, visit details, waiting times, admission status, referral information, and other operational metrics.


---

## 💡 Key Insights

- Identified peak emergency room visit periods.
- Compared admission versus non-admission trends.
- Analyzed patient waiting time patterns.
- Visualized patient distribution by age and gender.
- Evaluated referral patterns across departments.
- Enabled interactive filtering using slicers for faster analysis.
